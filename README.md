## 技术选型

### 前端

* Vue3

* Vant UI脚手架

* Vite脚手架

* Axios请求库


### 后端

* Java SpringBoot 2.7x

* MySQL

* MyBatis-Plus

* MyBatis X

* Redis缓存

* Redisson分布式锁

* Spring Scheduler定时任务

* 相似度匹配算法
